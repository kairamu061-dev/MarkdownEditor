# file-explorer テストケース

file-explorer は複数のサブ項目に分割されており、テストケースは各サブ項目の test-cases.md で管理する。
本エリア固有の横断的テストはなく、以下のサブ項目のケースで全体を担保する。

| サブ項目 | test-cases | 状態 |
|----------|-----------|------|
| vault | [file-explorer/vault/test-cases.md](./vault/test-cases.md) | 全合格 |
| tree-view | [file-explorer/tree-view/test-cases.md](./tree-view/test-cases.md) | 全合格（E-05 ダイアログ含む） |
| file-ops | [file-explorer/file-ops/test-cases.md](./file-ops/test-cases.md) | 全合格 |

## ステータス凡例

- `[ ]` 未実施
- `[~]` 実施中
- `[x]` 合格
- `[!]` 不合格

## E2Eテスト

| ID | テスト名 | 前提条件 | 手順 | 期待結果 | ステータス |
|----|---------|---------|------|---------|-----------|
| E-01 | 保管庫の切り替え | 2 つ以上の保管庫を開いた履歴あり | 保管庫名クリック → 別の保管庫を選択 | 選んだ保管庫のツリーに切り替わり、開いていたノートは閉じる | [ ] |
| E-02 | 履歴の永続化 | E-01 実施後 | 再起動して保管庫名クリック | 最近開いた保管庫が新しい順に一覧表示される | [ ] |
| E-03 | 別の保管庫を開く | スイッチャ表示中 | 「別の保管庫を開く…」→ フォルダ選択 | 選んだフォルダが開き、一覧の先頭に追加される | [ ] |

## ステータス凡例

- `[ ]` 未実施
- `[~]` 実施中
- `[x]` 合格
- `[!]` 不合格

実施記録: I-01 は 2026-07-08 に `npm run build` で合格。

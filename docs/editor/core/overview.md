# editor/core 概要

## 目的・背景

CodeMirror 6 によるエディタ本体を構築する。live-preview を含む今後のエディタ機能すべての基盤となる。

## スコープ

### 作るもの

- メイン領域へのエディタのマウントと破棄
- Markdown 言語サポート（GFM: 太字・斜体・打ち消し・コードブロック・リンク等）
- Nord 配色のエディタテーマ（背景・カーソル・選択・ガター無し）とシンタックスハイライト
- 標準編集キーマップ（undo/redo、インデント等）と行折り返し
- 公開 API: `mountEditor` / `getContent` / `setContent` / `onDocChanged`
- 起動時に開くスクラッチ文書（メモリ上、日本語の記法サンプル入り）

### 作らないもの

- 記法マークの表示/非表示（→ live-preview）
- ファイル入出力（→ file-explorer）

## 制約

- 拡張は `Extension` 配列として構成し、live-preview が外から追加できる構造にする
- 色は `nord.css` のエイリアスを参照（CodeMirror テーマ内でも `var(--...)` を使う）

## 完了条件

- アプリ起動時にメイン領域へエディタが表示され、スクラッチ文書を編集できる
- 見出し・強調・コード・引用・リスト・リンクが Nord 配色でハイライトされる
- `npm run build` と Xvfb 上の起動確認が通る

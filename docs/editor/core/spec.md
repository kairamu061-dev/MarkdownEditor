# editor/core 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | エディタ表示 | メイン領域全体にエディタを表示。最大幅 800px 中央寄せ、行折り返しあり |
| 2 | Markdown ハイライト | GFM 記法を Nord 配色で装飾（下表参照） |
| 3 | 標準編集操作 | 入力・選択・コピー/ペースト・undo (Ctrl+Z) / redo (Ctrl+Y, Ctrl+Shift+Z) |
| 4 | スクラッチ文書 | 起動時にメモリ上のサンプル文書を開く（保存されない） |
| 5 | 公開 API | file-explorer 接続用の内容取得・差し替え・変更通知 |

## ハイライト仕様（Nord 配色）

| 記法 | 表示 |
|------|------|
| 見出し 1〜6 | `--accent-secondary`（nord9）、太字、レベルに応じた文字サイズ（H1: 1.6em 〜 H6: 1.0em） |
| 太字 | `--text-strong`、太字 |
| 斜体 | 斜体 |
| 打ち消し | 打ち消し線、`--border` 色 |
| インラインコード / コードブロック | 等幅フォント、`--bg-hover` 背景 |
| リンク / URL | `--accent`（nord8）、下線 |
| 引用 | `--border` 色の左ボーダー、本文は `--nord4` |
| リストマーク | 記法マークと同色（アクセント色の弾丸表示は live-preview の置換で実現） |
| 記法マーク（`#` `**` 等） | `--border` 色（本文より沈む） |

## 画面・操作フロー

親 spec.md の「画面・操作フロー」1〜3 を参照（4〜5 は live-preview）。

## 各画面 / 状態の詳細

### エディタ（メイン領域）

- **表示内容**: スクラッチ文書の全文（Markdown ハイライト付き）
- **操作**: 機能 3 の編集操作
- **遷移先**: なし

## エラーケース

| 条件 | メッセージ / 挙動 |
|------|------------------|
| 不正な記法 | プレーンテキストとして表示（エラーなし） |

# editor/wikilink 概要

## 目的・背景

Obsidian の中核機能であるノート間リンク `[[ノート名]]` を提供する。
リンクとして整形表示し、クリックで該当ノートを開けるようにすることで、ヴォールトを「つながったノート群」として使えるようにする。

## スコープ

### 作るもの

- `[[ノート名]]` のリンク表示（アクセント色）とライブプレビュー（カーソル外では `[[ ]]` を隠す）
- 整形表示状態のリンクをクリックで該当ノートを開く（editor は解決をコールバックに委譲）
- ノート名の解決（file-explorer 側）: ヴォールト内から `ノート名.md` を大文字小文字無視で検索
- コードブロック・インラインコード内の `[[...]]` は対象外（リンク化しない）

### 作らないもの

- 入力補完（`[[` でノート名サジェスト）
- リンク先が存在しない場合のノート自動作成（Obsidian の挙動）— 見つからない旨をステータス表示
- リネーム時のリンク書き換え・バックリンク一覧
- `[[ノート名|表示名]]` のエイリアス記法・見出しリンク（`[[note#見出し]]`）

## 制約

- editor 側はヴォールトの知識を持たない（`onOpen(name)` コールバックで file-explorer に委譲）
- 文書テキストは変更しない（live-preview と同じ Decoration 方式）

## 完了条件

- `[[買い物リスト]]` がアクセント色リンクとして表示され、カーソル外では括弧が隠れる
- クリックで該当ノートが開き、ツリーの強調も追随する
- 存在しない名前のクリックでステータス表示（クラッシュ・無反応にならない）

## 画面イメージ

```
カーソル外:  関連: 買い物リスト     ← 下線+アクセント色、クリックで開く
カーソル内:  関連: [[買い物リスト]] ← 括弧が見え編集できる
```

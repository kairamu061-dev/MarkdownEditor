# editor/code-highlight 概要

## 目的・背景

フェンスコードブロック（```js 等）の中身を言語に応じてハイライトする。
技術メモ用途では コードの可読性が体験を大きく左右する。

## スコープ

### 作るもの

- 言語指定付きフェンスのコード内トークン着色（@codemirror/language-data による遅延読み込み）
- コード用トークンの Nord 配色（キーワード・文字列・コメント・数値・関数・型など）

### 作らないもの

- 言語指定なしフェンスの言語自動判定
- インラインコードのハイライト（等幅+背景のまま）
- 行番号・コピーボタン等のコードブロック UI

## 制約

- 言語パーサは使用時に動的 import される構成を保つ（起動時間・メモリへの影響を避ける。省メモリ方針）
- 色は nord.css のエイリアスと Nord パレット変数のみを使う

## 完了条件

- ```js / ```rust / ```python 等のコードが Nord 配色でトークン着色される
- 未知の言語指定はエラーにならず無着色（等幅+背景のみ）で表示される

# editor/code-highlight 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 言語別ハイライト | フェンスの言語指定（名前・エイリアス。js / typescript / rust / python / html / css / json 等）に応じてコード内を着色 |
| 2 | 遅延読み込み | 言語パーサは該当言語が最初に表示されたときに動的ロード |
| 3 | フォールバック | 未知の言語・指定なしは無着色（core の等幅+背景のみ） |

## コードトークンの Nord 配色

| トークン | 色 |
|----------|-----|
| キーワード | nord9（--accent-secondary） |
| 文字列 | nord14（--success） |
| コメント | nord3（--border）、斜体 |
| 数値・定数 | nord15 |
| 関数名 | nord8（--accent） |
| 型名・クラス名 | nord7 |
| プロパティ名 | nord4（--text） |
| 演算子・区切り | nord9 |

## 画面・操作フロー

1. ```js のようにフェンスへ言語を指定する
2. コードを入力すると（パーサ初回ロード後）トークンが着色される

## エラーケース

| 条件 | 挙動 |
|------|------|
| 未知の言語指定 | 無着色で表示（エラーなし） |
| パーサのロード失敗 | 無着色で表示（コンソールログのみ） |

## 未対応ケース

- 言語自動判定・シンタックスエラー表示
- コードブロック内の検索ハイライト強調

# editor 概要

## 目的・背景

本プロダクトの中核である Markdown 編集機能を提供する。
Obsidian のライブプレビュー（記法マークはカーソルが離れると隠れ、整形表示になる）を参考に、
CodeMirror 6 上で軽量に実現する。

## スコープ

### 作るもの

- CodeMirror 6 ベースの Markdown エディタコンポーネント（メインコンテンツ領域に表示）
- Nord 配色のエディタテーマと Markdown シンタックスハイライト
- ライブプレビュー（記法マークの表示/非表示切り替え）
- 後続エリア（file-explorer）から利用する公開 API（内容の取得・差し替え・変更通知）
- コードブロックの言語別シンタックスハイライト（2026-07-05 スコープ追加）
- ノート間リンク `[[wikilink]]` の表示とクリックでのオープン（2026-07-05 スコープ追加）

### 作らないもの

- ファイルの読み書き・保存（→ `file-explorer` エリアで保管庫管理と合わせて実装）
- リーディングビュー（完全な HTML プレビュー表示）
- 画像・埋め込みのレンダリング
- wikilink の補完・リネーム追随・バックリンク

## 制約

- エディタエンジンは CodeMirror 6 に限定（project_overview.md の技術選定に従う）
- 色は `nord.css` のエイリアスを参照する（ハードコード禁止）
- ファイル未接続の段階でも動作検証できるよう、起動時はメモリ上のスクラッチ文書を開く

## 完了条件

- アプリのメイン領域で Markdown 文書を編集でき、記法が Nord 配色でハイライトされる
- カーソルが記法範囲外にあるとき記法マークが隠れ、整形表示になる（ライブプレビュー）
- サブ項目（core / live-preview）の完了条件をすべて満たす

## 画面イメージ

```
+----------------+---------------------------------+
| [サイドバー]    | # 見出し (大きく表示)            |
|                |                                  |
|                | 本文テキスト **太字** ← カーソル  |
|                | が離れると「太字」のみ表示        |
|                |                                  |
|                | - リスト項目                     |
|                | > 引用                           |
+----------------+---------------------------------+
```

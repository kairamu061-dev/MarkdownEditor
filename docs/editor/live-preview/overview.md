# editor/live-preview 概要

## 目的・背景

Obsidian のライブプレビューを再現する。カーソルが記法範囲の外にあるときは記法マーク
（`**` `#` `` ` `` 等）を隠して整形表示し、カーソルが入ると記法マークを再表示して編集できるようにする。
「見た目はプレビュー、触れば即ソース」という Obsidian の中核体験にあたる。

## スコープ

### 作るもの

- 記法マークの隠蔽/再表示（見出し・太字・斜体・打ち消し・インラインコード）
- インラインリンクの整形表示（`[テキスト](URL)` → テキストのみ表示、URL と括弧を隠す）
- 箇条書きマーク（`-` `*` `+`）のアクセント色弾丸（•）への置換

### 作らないもの

- 引用の左ボーダー表示（`>` マークは常時表示のまま）
- 画像・埋め込みのレンダリング
- コードブロックフェンスの隠蔽
- リンクのクリックによる外部ブラウザ起動

## 制約

- core の `extraExtensions` 注入ポイントのみを使い、core のコードを変更しない（editor/design.md の横断事項）
- 表示切り替えは CodeMirror の Decoration（replace）で行い、文書テキスト自体は変更しない

## 完了条件

- カーソル外の記法マークが隠れ、カーソルを移動すると再表示される（対象: 見出し・太字・斜体・打ち消し・インラインコード・インラインリンク・箇条書きマーク）
- 編集操作（入力・undo/redo）が置換表示の影響を受けない
- Xvfb 上でカーソル内/外の 2 状態をスクリーンショットで確認できる

## 画面イメージ

```
カーソルが行外にあるとき          カーソルが太字部分にあるとき
--------------------------       --------------------------
見出し (大きく表示)               見出し (大きく表示)
本文の太字テキスト                本文の**太字|**テキスト
• リスト項目                      • リスト項目
リンクテキスト                    [リンクテキスト](https://…)
```

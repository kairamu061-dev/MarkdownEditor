# settings/ui 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | ⚙ ボタン | サイドバーヘッダ右端（[+] の隣）。押下でモーダルを開く |
| 2 | 設定モーダル | フォントサイズ（number、10〜32、既定 14）、フォントファミリー（select、プリセットから選択、既定=システムフォント） |
| 3 | 保存 | clamp(10〜32) して save_editor_settings。成功で即反映して閉じる |
| 4 | 破棄 | キャンセル / Esc / オーバーレイクリックで変更を破棄して閉じる |
| 5 | 起動時適用 | get_settings の editor 設定を CSS カスタムプロパティに反映 |

## 画面・操作フロー

親 spec.md の「画面・操作フロー」3〜5 を参照。

## 各画面 / 状態の詳細

### 設定モーダル

- **表示内容**: タイトル「設定」、フォントサイズ入力、フォントファミリー選択（select）、保存 / キャンセルボタン。エラー時はモーダル内に赤字 1 行
- **操作**: 入力・選択・保存・破棄
- **遷移先**: 閉じて通常画面

### フォントファミリーの選択肢（2026-07-12 自由入力から変更）

Windows 11 標準搭載フォントから構成。保存値はフォールバック付きの CSS `font-family` 文字列。

| 表示名 | 保存値 |
|--------|--------|
| システムフォント（既定） | null |
| 游ゴシック | `"Yu Gothic", "游ゴシック", sans-serif` |
| 游明朝 | `"Yu Mincho", "游明朝", serif` |
| メイリオ | `"Meiryo", "メイリオ", sans-serif` |
| BIZ UDゴシック | `"BIZ UDGothic", "BIZ UDゴシック", sans-serif` |
| BIZ UD明朝 | `"BIZ UDMincho", "BIZ UD明朝", serif` |
| 等幅（Consolas） | `Consolas, "BIZ UDGothic", monospace` |

- **後方互換**: 保存済みの fontFamily がプリセットのどの保存値とも一致しない場合（旧・自由入力の値）、
  「カスタム（現在の設定）」の選択肢を末尾に追加して選択状態にする。他の選択肢へ変更して保存した時点で消える

## フォント適用の仕様

- `--editor-font-family`: 未設定時はプロパティを外す（theme 側フォールバックでシステムフォント）
- `--editor-font-size`: `{n}px`。未設定時はプロパティを外す（フォールバック 14px）
- 適用対象はエディタ本文のみ（サイドバー等の UI フォントは変更しない）

## エラーケース

| 条件 | メッセージ / 挙動 |
|------|------------------|
| 保存失敗 | モーダル内に「設定の保存に失敗しました」。モーダルは開いたまま |
| サイズ範囲外入力 | 保存時に 10〜32 へ clamp |

## 未対応ケース

- 入力中のライブプレビュー（保存時にのみ反映）
- UI 全体のフォント設定

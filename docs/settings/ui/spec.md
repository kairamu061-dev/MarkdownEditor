# settings/ui 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | ⚙ ボタン | サイドバーヘッダ右端（[+] の隣）。押下でモーダルを開く |
| 2 | 設定モーダル | フォントサイズ（number、10〜32、既定 14）、フォントファミリー（text、空欄=システムフォント） |
| 3 | 保存 | clamp(10〜32) して save_editor_settings。成功で即反映して閉じる |
| 4 | 破棄 | キャンセル / Esc / オーバーレイクリックで変更を破棄して閉じる |
| 5 | 起動時適用 | get_settings の editor 設定を CSS カスタムプロパティに反映 |

## 画面・操作フロー

親 spec.md の「画面・操作フロー」3〜5 を参照。

## 各画面 / 状態の詳細

### 設定モーダル

- **表示内容**: タイトル「設定」、フォントサイズ入力、フォントファミリー入力（placeholder「システムフォント」）、保存 / キャンセルボタン。エラー時はモーダル内に赤字 1 行
- **操作**: 入力・保存・破棄
- **遷移先**: 閉じて通常画面

## フォント適用の仕様

- `--editor-font-family`: 未設定時はプロパティを外す（theme 側フォールバックでシステムフォント）
- `--editor-font-size`: `{n}px`。未設定時はプロパティを外す（フォールバック 14px）
- 適用対象はエディタ本文のみ（サイドバー等の UI フォントは変更しない）

## エラーケース

| 条件 | メッセージ / 挙動 |
|------|------------------|
| 保存失敗 | モーダル内に「設定の保存に失敗しました」。モーダルは開いたまま |
| サイズ範囲外入力 | 保存時に 10〜32 へ clamp |

## 未対応ケース

- 入力中のライブプレビュー（保存時にのみ反映）
- UI 全体のフォント設定

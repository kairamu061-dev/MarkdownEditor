# settings/ui 概要

## 目的・背景

設定を変更するためのモーダル UI を提供する。v1 の項目はエディタのフォントサイズとフォントファミリー。

## スコープ

### 作るもの

- サイドバーヘッダの ⚙ ボタンと設定モーダル（フォントサイズ・フォントファミリー）
- 起動時の設定読み込みとエディタへの適用（CSS カスタムプロパティ経由）
- 保存時の即時反映と永続化（editor 設定のみを更新する save_editor_settings コマンド）

### 作らないもの

- フォント以外の設定項目（テーマ・キーバインド等）
- フォント名の存在検証・プレビュー

## 制約

- フォントの適用は CSS カスタムプロパティ（--editor-font-family / --editor-font-size）経由とし、
  editor/core の theme.ts はこの変数を参照する形の最小変更に留める
- 保存は editor 設定のみを部分更新する（lastVault を上書きしない）

## 完了条件

- ⚙ → モーダルでフォントサイズ変更 → 保存 → エディタに即反映
- 再起動後もフォント設定が維持される

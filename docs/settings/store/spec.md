# settings/store 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | `get_settings` コマンド | settings.json を読み込んで返す。未存在・破損時はデフォルト値 |
| 2 | `save_settings` コマンド | 受け取った設定を pretty JSON で保存（親ディレクトリは自動作成） |
| 3 | `save_editor_settings` コマンド | editor フィールドのみ部分更新（ファイルを読み直して差し替え。lastVault を上書きしない。ui 用） |
| 4 | 内部ヘルパ | vault-restore が使う load / save 関数（パス指定可能でテスト可能） |

## 設定スキーマ

```json
{
  "lastVault": "C:\\Users\\me\\notes",   // 省略可。前回開いたヴォールトの絶対パス
  "editor": {
    "fontFamily": "Meiryo",              // 省略可。空・省略はシステムフォント
    "fontSize": 16                        // 省略可。省略時 14
  }
}
```

- 未知のフィールドは無視する（前方互換）
- 欠けたフィールドはデフォルト値で補う

## エラーケース

| 条件 | 挙動 |
|------|------|
| settings.json 未存在 | デフォルト値を返す |
| JSON 破損 | デフォルト値を返す（起動をブロックしない）。次回保存で上書き |
| 保存失敗（権限等） | エラーを返す（呼び出し側がステータス表示） |

## 未対応ケース

- スキーマバージョニング・マイグレーション（フィールド追加は serde default で吸収）

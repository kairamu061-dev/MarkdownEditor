- `[x]` 合格
- `[!]` 不合格

実施記録: I-01 / I-02 / E-01 / E-02 は 2026-07-04 に Linux devcontainer（Xvfb + xdotool）で実施し合格。
E-03 は 2026-07-05 に Windows 11 実機（dev モード）で確認し合格。

インストール起動と、マウスドラッグでの最小サイズ（640x480）制限は 2026-07-06 にユーザーが Windows 11 実機で
確認し合格。これで E-04 全体が合格。
E-05（フレームレス化＋自作タイトルバー）は 2026-07-12 にユーザーが Windows 11 実機で操作を確認し合格

なおフレームレス化で ☰ の開閉が配線切れになっていた（BUG-007、2026-07-12 ユーザー報告）。修正後、

# 用語集

このプロジェクトで使用するドメイン固有の用語を定義する。
実装・設計・仕様書内で用語を使う際はここに定義されたものに統一する。

---

| 用語 | 定義 |
|------|------|
| 保管庫 (Vault) | ユーザーが開くローカルフォルダ。配下の Markdown ファイル群を管理単位とする（Obsidian の Vault と同義）。コード上の識別子は `vault` |
| 保管庫スイッチャ | サイドバーの保管庫名から最近開いた保管庫の一覧を出し、その場で切り替える UI（Obsidian の Vault switcher 相当）。vault-switch サブ項目 |
| ノート | 保管庫内の 1 つの Markdown ファイル |
| ライブプレビュー | Markdown 記法を入力中はソースとして表示し、カーソルが離れると整形表示に切り替わる編集モード（Obsidian の Live Preview と同義） |
| ソースモード | Markdown を整形せず生テキストとして編集するモード |
| ファイルツリー | サイドバーに表示される保管庫内のフォルダ・ノートの階層表示 |
| Tauri コマンド | フロントエンドから IPC 経由で呼び出す Rust 側の関数 |
| Nord | 本プロジェクトが採用する寒色系カラーパレット（nord0〜nord15 の 16 色） |
| ウィキリンク (wikilink) | `[[ノート名]]` 記法によるノート間リンク。クリックで該当ノートを開く（Obsidian の内部リンクと同義） |
